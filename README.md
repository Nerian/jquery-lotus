# jquery-hanami

Jquery for Hanami.

This gem provides:

  * jQuery 1 and jQuery 2
  * the jQuery UJS adapter

## Installation

Add to Gemfile.
```
gem 'jquery-hanami'
```

Add to our tenmplate:
```
<%= javascript 'jquery' %>

or

<%= javascript 'jquery2' %>

optionally:
<%= javascript 'jquery_ujs' %>
```
